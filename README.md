# About me

Hi, I'm Tania — a town planner who got tired of waiting for better tools and decided to build them.
I've spent years working in low-mid rise town planning, complying development, and pre-construction documentation. Along the way I kept running into the same problem: the software just wasn't good enough. So I'm at Dev Academy learning to fix that.
I'm app-curious by nature — if I spot a problem, I'm already sketching a solution. My kids say I run over hurdles that are on fire, which feels about right.
I'm currently building an urban planning professionals portal — a workflow tool for architectural designers to manage jobs, documents, consultants, and council-facing compliance steps in one place. It's the project I always wished existed.
